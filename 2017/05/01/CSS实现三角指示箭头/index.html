    容易被忽略的一点

  • 伪类三角形的偏移值和矩形边框宽度一样吗？

答案是否定的！ 伪类三角形向下的偏移值其实应该是矩形边框宽度的”根号2”倍，约为1.414，为了方便可以按1.4倍计算。看下图是原理： 其中，效果中的矩形边框宽度为6px，所以要实现空心箭头边框宽度为6px的话，伪类三角形向下的偏移量应为6*1.4=8.4px
